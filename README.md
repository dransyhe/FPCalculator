# FPCalculator

A lexer and LR(0) parser supporting signed floating-point arithmetic expressions, with following operators:
 ```
"+": diadic infix, left-associative
"-": diadic infix, left-assoiative
"*": diadic infix, right-associative
"cos": monadic prefix
"!": monadic postfix
```
